隠しモード機能 - 動作確認テスト

テスト1: 通常アクセス（隠しモードOFF）

通常のURLでアクセスすると、ベータ機能ボタンは表示されません。

通常アクセスでテスト

期待される動作:

  • 「コンテンツをアップロード」ボタンが非表示
  • 「My Skill Sheet」ボタンが非表示
  • URL入力欄の下に何も表示されない

テスト2: 隠しモード（admin=true）

URLに ?admin=true を追加すると、ベータ機能が表示されます。

隠しモードでテスト

期待される動作:

  • 「コンテンツをアップロード BETA」ボタンが表示される
  • 「My Skill Sheet BETA」ボタンが表示される
  • 両ボタンをクリックすると、それぞれ /file?admin=true と /skill?admin=true に遷移する

テスト3: 無効なパラメータ

admin=false や admin=1 など、正確でない値では機能が有効にならないことを確認します。

admin=false でテスト admin=1 でテスト

期待される動作:

  • どちらの場合も、ベータ機能ボタンは非表示
  • admin=true のみが有効

実装詳細

隠しモード機能は以下のように実装されています:

  • トリガー: URLクエリパラメータ ?admin=true
  • 対象機能:
    • コンテンツをアップロード（/file）
    • My Skill Sheet（/skill）
  • デフォルト動作: ボタンは display: none で完全非表示
  • 遷移時の動作: admin モードは遷移先URLにも引き継がれる

使用方法

本番環境（Vercel）でも同様に動作します:

https://json-ld-view.vercel.app/?admin=true

このURLをブックマークしておくと、常に隠しモードでアクセスできます。
